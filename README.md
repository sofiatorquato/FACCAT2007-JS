# Exercícios de Lógica de Programação - FACCAT 2007

Repositório com a resolução de algoritmos em **JavaScript**, focados em lógica de programação e manipulação de variáveis.

## 📂 Estrutura dos Exercícios

| Arquivo | Problema Resolvido |
| :--- | :--- |
| `ex1.js` | Troca de valores entre duas variáveis (`A` e `B`). |
| `ex5.js` | Cálculo do antecessor de um número inteiro. |
| `ex6.js` | Cálculo da área de um retângulo (base × altura). |
| `ex7.js` | Conversão de idade (anos, meses e dias) para total de dias. |
| `ex8.js` | Cálculo de percentual de votos (brancos, nulos e válidos). |
| `ex9.js` | Cálculo de reajuste salarial a partir de um percentual. |
| `ex10.js` | Cálculo do custo final ao consumidor de um automóvel. |

## 🛠️ Tecnologias
* **Linguagem:** JavaScript
* **Execução:** Console do navegador
